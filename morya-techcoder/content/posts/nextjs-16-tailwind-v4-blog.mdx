## Why Next.js 16 + Tailwind v4?

The combination of Next.js 16's improved App Router and Tailwind v4's new Oxide engine is the fastest way to build a production blog in 2026.

### What's New in Tailwind v4

Tailwind v4 dropped the `tailwind.config.js` file in favor of CSS-first configuration:

```css
@import "tailwindcss";

@theme inline {
  --color-brand: #FFB433;
  --font-heading: 'Montserrat', sans-serif;
}
```

This is a massive DX improvement. Your design tokens live right next to your styles.

### Setting Up the Project

```bash
npx create-next-app@latest my-blog --app --tailwind --typescript
```

### The App Router Advantage

With the App Router, each route is a folder:

```
app/
  page.tsx          # Home
  blog/
    page.tsx        # Blog list
    [slug]/
      page.tsx      # Blog detail
```

### Performance Tips

- Use `generateStaticParams` for pre-rendering blog posts
- Leverage the built-in Image component for automatic optimization
- Keep client components minimal — most blog content should be server-rendered

### Result

A blog that scores 100 on Lighthouse, loads in under 1 second, and is a joy to maintain. What more could you ask for?
