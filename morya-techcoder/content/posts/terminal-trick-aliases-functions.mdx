## Stop Typing the Same Commands

If you type `git add . && git commit -m "..." && git push` more than once a day, you're wasting time.

### Essential Aliases

Add these to your `.zshrc` or `.bashrc`:

```bash
# Git shortcuts
alias gs="git status"
alias gp="git push"
alias gl="git log --oneline -20"
alias gc="git commit -m"
alias gac="git add . && git commit -m"

# Navigation
alias ..="cd .."
alias ...="cd ../.."
alias dev="cd ~/Developer"

# Development
alias nr="npm run"
alias nrd="npm run dev"
alias nrb="npm run build"
```

### Power Functions

Aliases are great for simple commands. Functions handle the complex ones:

```bash
# Create a new branch and switch to it
nb() { git checkout -b "$1"; }

# Quick commit with message
qc() { git add . && git commit -m "$1" && git push; }

# Kill process on a port
killport() { lsof -ti:"$1" | xargs kill -9; }

# Create directory and cd into it
mkcd() { mkdir -p "$1" && cd "$1"; }
```

### FZF: The Game Changer

Install `fzf` for fuzzy finding everything:

```bash
# Fuzzy find and open files
alias f="fzf --preview 'cat {}' | xargs code"

# Fuzzy find git branches
alias gb="git branch | fzf | xargs git checkout"
```

> The best developers aren't faster typists — they're smarter about what they type.

### Getting Started

1. Open your shell config: `code ~/.zshrc`
2. Paste the aliases above
3. Run `source ~/.zshrc`
4. Enjoy your extra 30 minutes every day
